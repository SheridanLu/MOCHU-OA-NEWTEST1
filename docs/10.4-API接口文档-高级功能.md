## 28.3 创建现场签证

### 请求
```
POST /api/change/visa
Authorization: Bearer <token>
Content-Type: application/json
```

### 权限要求
`change:create`

### 请求体
```json
{
  "project_id": 1,
  "visa_type": "additional_work",
  "content": "增加电缆敷设50米",
  "reason": "设计变更要求",
  "amount": 5000.00,
  "attachments": [
    {
      "name": "签证单.pdf",
      "url": "/uploads/visa.pdf"
    }
  ]
}
```

### 参数说明
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| project_id | number | 是 | 项目ID |
| visa_type | string | 是 | additional_work, design_change, other |
| content | string | 是 | 签证内容 |
| reason | string | 是 | 签证原因 |
| amount | number | 否 | 涉及金额 |
| attachments | array | 否 | 附件列表 |
