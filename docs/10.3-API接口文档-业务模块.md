```json
    {

      "specification": "3*2.5mm²",
      "unit": "米",
      "quantity": 1000,
      "unit_price": 15.00
    }
  ]
}
```

### 响应
```json
{
  "success": true,
  "message": "批量采购创建成功",
  "data": {
    "id": 5,
    "batch_no": "BP-2026-005",
    "status": "draft"
  }
}
```

---

## 13.3 批量采购详情

### 请求
```
GET /api/purchase/batch/batch/:id
Authorization: Bearer <token>
```

---

## 13.4 更新/删除/提交/审批

```
PUT /api/purchase/batch/batch/:id      # 更新
DELETE /api/purchase/batch/batch/:id   # 删除
POST /api/purchase/batch/batch/:id/submit   # 提交
POST /api/purchase/batch/batch/:id/approve  # 审批通过
```

# 14. 零星采购接口 (/api/sporadic-purchase)

## 14.1 零星采购列表

### 请求
```
GET /api/sporadic-purchase?page=1&pageSize=20
Authorization: Bearer <token>
```

### 响应
```json
{
  "success": true,
  "data": {
    "list": [
      {
        "id": 1,
        "sporadic_no": "SP-2026-001",
        "project_id": 1,
        "project_name": "某某项目",
        "contract_id": 2,
        "contract_no": "INC-2026-002",
        "total_amount": 5000.00,
        "tax_rate": 13,
        "status": "approved",
        "creator_name": "张三"
      }
    ],
    "total": 15
  }
}
```

---

## 14.2 创建零星采购

### 请求
```
POST /api/sporadic-purchase
Authorization: Bearer <token>
Content-Type: application/json
```

### 权限要求
`material:create`

### 请求体
```json
{
  "project_id": 1,
  "contract_id": 2,
  "tax_rate": 13,
  "remark": "紧急采购",
  "items": [
    {
      "item_name": "电缆线",
      "specification": "3*2.5mm²",
      "unit": "米",
      "quantity": 100,
      "unit_price": 15.00
    }
  ],
  "attachments": [
    {
      "name": "报价单.pdf",
      "url": "/uploads/quote.pdf"
    }
  ]
}
```

### 参数说明
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| project_id | number | 是 | 项目ID |
| contract_id | number | 是 | 收入合同ID（用于校验金额不超过0.5%） |
| tax_rate | number | 否 | 税率（0, 1, 3, 6, 9, 13） |
| items | array | 是 | 采购项列表 |
| attachments | array | 否 | 附件列表 |

### 验证规则
- 零星采购金额不能超过收入合同金额的0.5%

---

## 14.3 检查超额

### 请求
```
POST /api/sporadic-purchase/check-excessive
Authorization: Bearer <token>
Content-Type: application/json
```

### 请求体
```json
{
  "contract_id": 2,
  "amount": 5000.00
}
```

### 响应
```json
{
  "success": true,
  "data": {
    "is_excessive": false,
    "contract_amount": 1000000.00,
    "used_amount": 3000.00,
    "remaining_amount": 997000.00,
    "threshold_amount": 5000.00
  }
}
```
