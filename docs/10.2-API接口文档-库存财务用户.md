# 5. 库存管理接口 (/api/stock)

## 5.1 材料基准价列表

### 请求
```
GET /api/materials/base?page=1&pageSize=20&keyword=电缆
Authorization: Bearer <token>
```

### 查询参数
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| keyword | string | 否 | - | 搜索关键词（名称/规格） |
| status | string | 否 | - | active, inactive |
| page | number | 否 | 1 | 页码 |
| pageSize | number | 否 | 20 | 每页数量 |

### 响应
```json
{
  "success": true,
  "data": {
    "list": [
      {
        "id": 1,
        "name": "电缆线",
        "specification": "3*2.5mm²",
        "unit": "米",
        "base_price": 14.00,
        "warning_threshold": 1,
        "status": "active",
        "created_at": "2026-01-01 10:00:00"
      }
    ],
    "total": 100
  }
}
```

---

## 5.2 添加材料基准价

### 请求
```
POST /api/materials/base
Authorization: Bearer <token>
Content-Type: application/json
```

### 权限要求
`material:create`

### 请求体
```json
{
  "name": "电缆线",
  "specification": "3*2.5mm²",
  "unit": "米",
  "base_price": 14.00,
  "warning_threshold": 1
}
```

### 参数说明
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 材料名称，最长100字符 |
| specification | string | 否 | 规格型号，最长100字符 |
| unit | string | 是 | 单位 |
| base_price | number | 是 | 基准价（含税），≥0 |
| warning_threshold | number | 否 | 预警阈值(%)，0-100 |

### 响应
```json
{
  "success": true,
  "message": "材料基准价添加成功",
  "data": {
    "id": 50,
    "name": "电缆线",
    "base_price": 14.00
  }
}
```

## 5.4 创建入库单

### 请求
```
POST /api/stock/in
Authorization: Bearer <token>
Content-Type: application/json
```

### 权限要求
`inventory:in`

### 请求体
```json
{
  "project_id": 1,
  "supplier_id": 1,
  "warehouse": "主仓库",
  "remark": "备注",
  "items": [
    {
      "material_id": 1,
      "material_name": "电缆线",
      "specification": "3*2.5mm²",
      "unit": "米",
      "quantity": 1000,
      "unit_price": 15.00,
      "total_price": 15000.00
    }
  ]
}
```

### 参数说明
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| project_id | number | 是 | 项目ID |
| supplier_id | number | 否 | 供应商ID |
| warehouse | string | 否 | 仓库 |
| remark | string | 否 | 备注 |
| items | array | 是 | 入库明细，至少1项 |

### items结构
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| material_id | number | 否 | 材料基准价ID |
| material_name | string | 是 | 材料名称 |
| specification | string | 否 | 规格型号 |
| unit | string | 是 | 单位 |
| quantity | number | 是 | 数量，>0 |
| unit_price | number | 是 | 单价，≥0 |
| total_price | number | 是 | 总价 |

### 响应
```json
{
  "success": true,
  "message": "入库单创建成功",
  "data": {
    "id": 15,
    "stock_in_no": "IN-2026-015",
    "total_amount": 15000.00,
    "status": "confirmed"
  }
}
```


### 响应
```json
{
  "success": true,
  "data": {
    "list": [
      {
        "id": 1,
        "material_name": "电缆线",
        "specification": "3*2.5mm²",
        "base_price": 14.00,
        "purchase_price": 15.50,

        "warning_level": "serious",
        "project_name": "某某项目",
        "status": "pending"
      }
    ],
    "total": 5
  }
}
```

### 预警级别
- `warning`: 偏差率 1% - 3%
- `serious`: 偏差率 > 3%

## 7.2 创建用户

### 请求
```
POST /api/users
Authorization: Bearer <token>
Content-Type: application/json
```

### 权限要求
`user:create`

### 请求体
```json
{
  "username": "lisi",
  "password": "123456",
  "real_name": "李四",
  "phone": "13900139000",
  "email": "lisi@mochu.group",
  "department_id": 1,
```

### 参数说明

| real_name | string | 是 | 真实姓名，2-20字符 |
| phone | string | 否 | 手机号，11位 |
| email | string | 否 | 邮箱 |
| department_id | number | 否 | 部门ID |
| role_ids | array | 否 | 角色ID数组 |

### 响应
