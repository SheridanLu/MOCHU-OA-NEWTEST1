## 33.3 创建任务依赖

### 请求
```
POST /api/construction/dependencies
Authorization: Bearer <token>
Content-Type: application/json
```

### 请求体
```json
{
  "predecessor_id": 2,
  "successor_id": 1,
  "dependency_type": "FS",
  "lag_days": 0
}
```

### 参数说明
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| predecessor_id | number | 是 | 前置任务ID |
| successor_id | number | 是 | 后继任务ID |
| dependency_type | string | 是 | FS(完成-开始), SS(开始-开始), FF(完成-完成), SF(开始-完成) |
| lag_days | number | 否 | 滞后天数 |

### 响应
```json
{
  "success": true,
  "message": "任务依赖创建成功",
  "data": {
    "id": 5
  }
}
```

# 34. 库存查询高级功能

## 34.1 库存查询

### 请求
```
GET /api/stock/query?project_id=1&keyword=电缆
Authorization: Bearer <token>
```

### 查询参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| project_id | number | 否 | 项目ID |
| keyword | string | 否 | 搜索关键词 |
| category | string | 否 | 分类 |
| location | string | 否 | 库存位置 |
| low_stock | boolean | 否 | 仅显示低库存 |

### 响应
```json
{
  "success": true,
  "data": {
    "list": [
      {
        "id": 1,
        "material_name": "电缆线",
        "specification": "3*2.5mm²",
        "unit": "米",
        "total_quantity": 5000,
        "available_quantity": 3000,
        "locked_quantity": 500,
        "used_quantity": 1500,
        "location": "A仓库",
        "warning_threshold": 1000,
        "is_low_stock": false
      }
    ],
    "total": 50
  }
}
```

# 附录I：任务依赖类型说明

| 类型 | 名称 | 说明 |
|------|------|------|
| FS | 完成-开始 | 前置任务完成后，后继任务才能开始 |
| SS | 开始-开始 | 前置任务开始后，后继任务才能开始 |
| FF | 完成-完成 | 前置任务完成后，后继任务才能完成 |
| SF | 开始-完成 | 前置任务开始后，后继任务才能完成 |

---

# 附录J：库存状态说明

| 状态 | 说明 |
|------|------|
| available | 可用库存 |
| locked | 锁定库存（出库申请中） |
| used | 已使用库存 |
| total | 总库存 = available + locked + used |

---

# 附录K：预警类型说明

| 预警类型 | 说明 |
|---------|------|
| over_limit | 超过限额 |
| low_stock | 库存不足 |
| price_deviation | 价格偏差 |
| progress_delay | 进度延期 |
| milestone_overdue | 里程碑超期 |
